Request form | Request | BODY | 1 4 each request contains of 5 bytes 1^st byte represents request type following 4 bytes represent additional request data. Some requests will use this space, some will leave it empty. For example: Request type: CONNECT_SENSOR CONNECT_SENSOR.byteValue, SENSOR_TYPE.byteValue, -, -, - Request type: CONFIGURE CONFIGURE.byteValue, 0, 0, 0, 0b0000_1111 - informs that maximum amount of data it can get is 15 bytes Request type: IS_CONNECTED IS_CONNECTED.byteValue, -, -, -, - etc.
